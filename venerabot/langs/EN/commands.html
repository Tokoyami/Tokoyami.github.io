List of commands:
+help - show this list
            
+ping - ping of the bot
            
+sum - sum of the numbers you wrote
            
+creator - bot Creator
            
+kick - kicks the mentioned user
            
+ban - bans the mentioned user
            
+members - Number of participants on the server
            
+eject - throw out the mentioned user (among us)
            
+cat - random cat
            
+search - Searches in Yandex.Query
            
+urban - searches the urban dictionary for the meaning of the word
            
+play (url) - play music from youtube
            
+stop - stop music
            
+fox - random Fox
            
+version - information about the bot's version
            
+giveaway (time in hours) (reward) - make a giveaway!
